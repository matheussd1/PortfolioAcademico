# Portfólio Acadêmico - Matheus Dias

Este projeto é um portfólio pessoal dinâmico desenvolvido como parte de uma avaliação prática de desenvolvimento web. O foco foi criar uma interface **minimalista, artística e funcional**, utilizando tecnologias modernas de front-end e boas práticas de UX/UI.

---
## De uma olhada no projeto: 
https://matheussd1.github.io/PortfolioAcademico/

## 🛠️ Tecnologias Utilizadas

O projeto foi construído "do zero" (Vanilla), sem o uso de frameworks pesados, para demonstrar domínio sobre os fundamentos da web:

- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Layouts com Flexbox, animações personalizadas e estilização *Dark Mode*.
- **JavaScript (ES6+)**: Manipulação dinâmica do DOM, renderização de dados a partir de objetos e lógica de interatividade.
- **Git/GitHub**: Controle de versão e deploy automatizado via GitHub Pages.

---

## ✨ Funcionalidades em Destaque

- **Navbar Animada**: Fundo em vídeo abstrato com sobreposição de camadas e efeitos de *hover* dinâmicos.
- **Saudação Inteligente**: O sistema identifica o horário do usuário e gera uma saudação personalizada (Bom dia/Boa tarde/Boa noite).
- **Timeline Acadêmica**: Seção de formação e cursos complementares com design de linha do tempo e interatividade lateral.
- **Gestão Dinâmica de Projetos**: Os projetos são carregados via JavaScript, facilitando a manutenção e garantindo a separação entre dados e interface.
- **Cópia de E-mail Inteligente**: Botão de contato que copia o endereço de e-mail para a área de transferência com feedback visual instantâneo.
- **Design Responsivo**: Adaptado para uma experiência fluida em computadores, tablets e smartphones.
